## Instructions

1. To get started, run code 
2. Enter a "1", "2", or "3" to enter the prompted room 
3. Go to lobby to catch a breather 
4. Go to shop to acquire legendary card 
5. Enter tournament to compete
    * If legendary card present, you win
    * If no legendary card present, you lose
6. Game restarts regardless of win or lose
